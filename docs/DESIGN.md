# 디자인 (DESIGN)

## 1. 디자인 목표

- **원팀 몰입** — 앱에 들어오면 "여긴 LG 트윈스 팬의 공간". 팀 정체성이 색·톤에 흐른다.
- **코인/시세 경험** — 선수 가치를 주식·코인 시세처럼 느끼게. 떡상/떡락이 직관적으로 읽혀야 한다.
- **데이터 가독성 · 모바일 우선** — 핵심은 그래프. 좁은 화면·엄지 조작 기준으로 결정한다.

## 2. 브랜드 (LG 트윈스)

- 상단은 **LG 레드 full-bleed 헤더**로 화면 끝까지 채워 "전체 화면을 쓰는" 인상을 준다
  (배경색은 100% 폭, 콘텐츠는 넓은 컨테이너).
- 레드를 포인트로 화이트 베이스에 절제 사용. 데이터 그래프의 가독성을 해치지 않는 선에서.

## 3. 시그니처 — 코인 캔들 차트

이 앱의 정체성은 선수 가치를 **코인 시세 캔들**로 보여주는 것이다.

- **떡상 = 빨강 / 떡락 = 파랑** 캔들. 캔들 하나 = 그 경기의 등락(가치 기여).
- **이동평균선**(주황) = 가치 추세(진짜 신호). **시작가 기준선** = 코인 base 100.
- **일별 / 주별 / 월별 토글**(클라이언트 리샘플), 호버 툴팁.
- 색 기준(수면선) = "평균 경기(가치 0)". 캔들 크기 = 그날 가치 크기.
- (향후) 손가락 패닝·핀치 줌 — 현재는 리샘플 토글로 구간 전환, 제스처 줌은 다음 단계.

## 4. 홈 — 트윈스 코인 시세판

- 타자·투수 코인 **시세 랭킹** 카드 그리드. 각 카드에 현재 코인가 + **미니 시세선**.
- 타자/투수를 **배지로 구분**(색 구분). 데스크톱 4열 / 모바일 2열 반응형.
- full-bleed 헤더 + 넓은 컨테이너로 화면을 채운다.

## 5. 투명성 UI

코인은 박스스코어 기반 **근사 지표(WAR 아님, 수비·클러치 제외)**임을 상시 노출한다. 과신을
막고 신뢰를 주는 장치이자, 분석 태도를 보여주는 디자인 결정이다.

## 6. 정보 구조

- **홈** — 트윈스 코인 시세판(랭킹 + 미니 시세선).
- **선수** — 선수 선택 → 코인 캔들 차트(코인/타율·OPS 토글, 타자/투수 필터).
- **뉴스** — LG 뉴스 목록 → 원문 링크.
- (로드맵) **이동** — 등록/말소·트레이드 + 사유.

## 7. 향후

- 캔들 제스처 줌(lightweight-charts 등), 팀 순위 히어로 밴드, 다크 모드, 색 팔레트·컴포넌트 토큰 정리.
